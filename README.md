## setup

you need node `22.12.0` or newer and `pnpm`.

clone the repo:

```sh
git clone https://github.com/sebashtioon/new-sebashtioon-dev.git
cd new-sebashtioon-dev
```

install everything:

```sh
pnpm install
```

run it locally:

```sh
pnpm dev
```

then open the local astro dev server, usually `http://localhost:4321`.

build the site:

```sh
pnpm build
```

preview the build:

```sh
pnpm preview
```
