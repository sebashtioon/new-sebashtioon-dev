# new-sebashtioon-dev

personal portfolio website built with React and Vite. https://sebashtioon.dev

## features

- responsive layout for desktop & mobile  
- light/dark theme support  
- modular component structure for easy updates  

## running locally

1. clone the repo:  
```bash
git clone https://github.com/sebashtioon/new-sebashtioon-dev.git
cd new-sebashtioon-dev
```

2. install dependencies:  
```bash
npm install
```

3. start the dev server:  
```bash
npm run dev
```

4. open the url shown in the terminal
